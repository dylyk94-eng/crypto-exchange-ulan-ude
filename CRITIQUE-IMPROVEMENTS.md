# CryptoX — Улучшения v3.3 на основе критики 🎯

## 📊 Анализ проблем

### Критические проблемы (Высокий приоритет)

#### 1. 🖼️ Отсутствие изображений
**Проблема:** Все placeholder изображения не существуют в `public/`
**Влияние:** Пустые карточки, недоверие к сайту
**Решение:** Создан компонент `ImagePlaceholder.tsx` с SVG placeholder

**Реализация:**
```tsx
<ImagePlaceholder type="office" width={400} height={300} />
```

---

#### 2. 📊 Перегруз информацией
**Проблема:** 14 секций на главной странице
**Влияние:** Информационная перегрузка, сложно найти ключи
**Решение:** Сокращено до 7 секций

**Новая структура:**
```
1. Hero (Bento grid)
2. Section Separator
3. Города (2 карточки)
4. Section Separator
5. Как это работает (4 шага)
6. Section Separator
7. Услуги (compact)
8. Section Separator
9. Contact (упрощенная форма)
10. Footer
```

---

#### 3. 🎨 Проблемы с контрастностью
**Проблема:** `text-white/70` плохо читаем на тёмном фоне
**Влияние:** Плохая читаемость, accessibility issues
**Решение:** Увеличено до `text-white/80` и `text-white/90`

**Изменения в CSS:**
```css
:root {
  --text-secondary: rgba(255, 255, 255, 0.8); /* было 0.7 */
  --text-tertiary: rgba(255, 255, 255, 0.6); /* было 0.5 */
}
```

---

#### 4. 📝 Длинные формы
**Проблема:** 6 полей отпугивают пользователей
**Влияние:** Низкая конверсия, утомляет
**Решение:** Сокращено до 3 обязательных + 1 необязательное

**Новая форма:**
```
Обязательные:
1. Имя
2. Телефон
3. Валюта

Необязательное:
4. Сумма
```

**Убраны поля:**
- Telegram (перенесён в CTA)
- Комментарий

---

### Средние проблемы (Средний приоритет)

#### 5. ✨ Отсутствие визуальных разделителей
**Проблема:** Все секции выглядят одинаково
**Влияние:** Нет визуальной иерархии, сложно разделить контент
**Решение:** Создан компонент `SectionSeparator.tsx`

**Варианты:**
```tsx
<SectionSeparator type="gradient" />  // Линия с градиентом
<SectionSeparator type="dashed" />   // Пунктирная линия
<SectionSeparator type="dotted" />   // Точечная линия
```

---

#### 6. 🎯 Отсутствие clear CTA
**Проблема:** Много кнопок "Связаться", не понятно главное действие
**Влияние:** Пользователь путается в действиях
**Решение:** Выделен PRIMARY CTA "Отправить заявку"

**CTA иерархия:**
```
PRIMARY: "Отправить заявку" (в форме)
SECONDARY: "Написать в Telegram" (в форме)
NAVIGATION: "Выбрать город", "Оставить заявку"
```

---

## 📦 Новые компоненты

### ImagePlaceholder.tsx
Универсальный placeholder для отсутствующих изображений.

**Props:**
```tsx
<ImagePlaceholder
  type="city" | "office" | "service" | "avatar"
  width={400}
  height={300}
  icon="🏙️"
  title="Улан-Удэ"
/>
```

**Особенности:**
- ✅ Pattern overlay для визуального интереса
- ✅ Градиентный фон по типу
- ✅ Иконка в центре
- ✅ Индикатор "Image" с loading pulse
- ✅ Gradient border
- ✅ Accessible (role="img", aria-label)

---

### SectionSeparator.tsx
Визуальные разделители между секциями.

**Варианты:**
```tsx
<SectionSeparator type="gradient" />  // Градиентная линия
<SectionSeparator type="dashed" />   // Пунктирная линия
<SectionSeparator type="dotted" />   // Точечная линия
```

**Особенности:**
- ✅ Градиент от transparent до transparent
- ✅ Пунктирная линия
- ✅ Точечная линия
- ✅ Кастомный className
